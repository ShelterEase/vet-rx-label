🖨 How to print — 3 steps
1.Fill out all fields → click Copy label text
2.Switch to DYMO Connect → right-click inside the text box → Paste (or Ctrl+A then Ctrl+V to replace existing text)
3.Click Print in DYMO Connect — done!
⚙ One-time DYMO Connect setup
1.Open DYMO Connect → New Label
2.Select 30324 Large Multipurpose
3.Click the Text tool and drag a box that fills the entire label
4.In the text box properties, set font to Courier New, Bold, 8pt
5.File → Save As → name it VetRx → save to Desktop
6.Next time: just double-click VetRx.label on Desktop to open it ready to paste
